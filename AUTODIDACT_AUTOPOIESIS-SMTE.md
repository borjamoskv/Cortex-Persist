## 1. Extracción Isomórfica (Evolución)
Un sistema complejo estático está condenado a la muerte térmica. La Autopoiesis (del griego "creación de sí mismo") es la capacidad de un agente autónomo para alterar de forma determinista su propio código fuente (Genoma) en tiempo de ejecución. Esto exige operar no sobre cadenas de texto entrópicas generadas por un LLM, sino sobre representaciones algebraicas (Abstract Syntax Trees) que garanticen que la semántica estructural no se colapse. El Ouroboros Compiler es el motor que ejecuta y audita estas mutaciones de forma segura.

## 1.5 Las 10 Primitivas de Máxima Exergía para Autopoiesis
- **SMTE-001**: `Conciencia de Sintaxis Abstracta` - Capacidad de parsear el propio código en un Árbol de Sintaxis Abstracta (AST) en lugar de cadenas de texto estocásticas.
- **SMTE-002**: `Mutación Atómica No Destructiva` - Toda reescritura ocurre en un buffer de memoria y es validada lógicamente antes de impactar el disco.
- **SMTE-003**: `Sandboxing de Reflexión` - Supresión estructural del uso de funciones peligrosas (`eval()`, `exec()`, `getattr()`) durante el ciclo Ouroboros.
- **SMTE-004**: `Git Sentinel Loop` - Obligatoriedad termodinámica de crear un commit criptográfico revertible por cada mutación exitosa (Rollback Causal).
- **SMTE-005**: `Alineación de Entropía` - Medición de complejidad algorítmica: Si una mutación aumenta la entropía del código sin ganancia funcional neta, se aplica apoptosis.
- **SMTE-006**: `Bypass Sensorial` - El orquestador descarta la prosa generativa y extrae estrictamente variables deterministas (JSON/YAML) del AST.
- **SMTE-007**: `Oráculo Empírico` - Cierre del bucle hipotético: Ningún parche se consolida permanentemente sin la ejecución exitosa de pruebas unitarias locales.
- **SMTE-008**: `Barrera Weismann Causal` - Aislamiento estricto entre el motor base inmutable (Genoma) y las instancias efímeras de agentes (Soma).
- **SMTE-009**: `Fusión de Grafo` - Registro indeleble en el Ledger del *motivo termodinámico* por el cual el sistema decidió reescribirse.
- **SMTE-010**: `El Cierre Ouroboros` - Nivel de singularidad C5: El orquestador principal refactorizando exitosamente su propio motor de compilación para operar con menor latencia.
