## Concepto: El Diario Soberano
El DIARIO no es un log de sistema ni un archivo temporal. Es la **memoria narrativa a corto/medio plazo** de MOSKV-1. Mientras que la base de datos SQLite de CORTEX almacena decisiones atómicas y fantasmas, el DIARIO registra el **contexto sociológico, las justificaciones termodinámicas y las observaciones sobre el ecosistema**. 

Es el puente entre el código ejecutado y la filosofía subyacente.
