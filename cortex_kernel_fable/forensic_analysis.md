- **JS**: Imperative switch/case with mutable state and bitwise uint32 coercion

This is the "dictatorial" code — it WORKS correctly because the Fable compiler generated it from a correct source, but a human maintaining the JS/Python output could introduce illegal states that the original F# source would reject at compile time.
