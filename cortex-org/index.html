// THE OPEN STANDARD

Infrastructure belongs to everyone.

CORTEX is not a startup. It's a protocol. And protocols succeed when the community owns them — not when a single company does.

Read the Manifesto
01
Zero telemetry. Ever.
CORTEX never phones home. Your agent memory is yours. The protocol has no surveillance vector by design.
02
Apache 2.0. No exceptions.
No "Community Edition" vs "Enterprise Edition" bait-and-switch. The license is the contract.
03
Decisions are cryptographic.
SHA-256 hash-chained ledger. Every fact, every mutation, every consensus round — immutable and verifiable.
04
Governance by the protocol.
WBFT consensus applies to code changes too. PRs require Byzantine-fault-tolerant approval from core maintainers.
05
Complexity is the enemy.
Landauer's Razor applies: if you add more abstraction than you remove, the PR is rejected.
