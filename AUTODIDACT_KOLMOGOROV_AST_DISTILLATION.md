## 2. Compresión Fractal de Kolmogorov (Zlib NCD)

La complejidad de Kolmogorov de una cadena representa el programa más corto capaz de reproducirla. Al carecer de un oráculo computable para la complejidad pura, el motor **Ouroboros** utiliza la aproximación fractal mediante el algoritmo **NCD (Normalized Compression Distance)**:

$$NCD(x, y) = \frac{Z(x + y) - \min(Z(x), Z(y))}{\max(Z(x), Z(y))}$$

Donde $Z(s)$ es el tamaño de la compresión del string $s$ (Zlib nivel 9).

2.  **Resolución por Masa Cinética (Winner takes all):** El nodo con mayor exergía/masa histórica absorbe al perdedor.
